TO GET STARTED

1. cd functions

2. npm i

3. create 3 new files under functions root - .env, .env.development, .env.production

4. add env variables

   for .env
   NODE_ENV="development"
   SIGNATURE=
   APIKEY=
   AUTHDOMAIN=
   PROJECTID=
   STORAGEBUCKET=
   MESSAGINGSENDERID=
   APPID=
   EMAILUSER=
   EMAILPASS=

   for .env.development

   for .env.production

5. add service account private key under functions root

6. npm start
